Selain location_id, Anda juga bisa meminta data berdasarkan district_code atau code.

Endpoint Lainnya

Selain endpoint locations dan clinics, Anda juga dapat meminta data dari endpoint lain. Berikut daftar endpoint lain beserta query yang tersedia:

  • clinics_holidays

    Query yang tersedia: clinic_id, holiday_name.

  • clinics_services

    Query yang tersedia: clinic_id, service_id.

  • clinics_work_times

    Query yang tersedia: clinic_id, day_num.

Cara meminta data untuk semua, satu data, atau berdasarkan query tertentu sama seperti yang telah dijelaskan sebelumnya. Silakan periksa dokumentasi di atas.

Meminta Data Reservasi

Jika Anda berencana membangun halaman frontend untuk reservasi dan ingin mendapatkan data klinik untuk ditampilkan atau dipilih oleh pengguna, Anda dapat merujuk ke bagian sebelumnya tentang cara mendapatkan data klinik berdasarkan lokasi. Pada bagian ini, kita akan fokus pada informasi waktu operasional untuk setiap klinik.

Gambar di atas menunjukkan data waktu operasional klinik berdasarkan tanggal saat ini dan ke depan. Untuk mendapatkan data ini, kirimkan beberapa query seperti clinic_id dan interval (durasi gap waktu pecahan).

https://beranibersama.luth.my.id/reservasi-time?clinic_id=309&interval=30
